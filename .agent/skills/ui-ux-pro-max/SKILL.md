# UI/UX Pro Max

## Core Principles
- **Bold Intentionality**: Avoid generic "safe" designs. Commit to a clear aesthetic (Glassmorphism, Flat Modern, etc.).
- **Visual Hierarchy**: Use contrast, size, and weight to guide the eye to primary actions.
- **Premium Aesthetics**: Use custom gradients, subtle noise textures, and layered transparencies.
- **Micro-interactions**: Every action should have a smooth, tactile feedback (hover scales, active presses, staggered reveals).

## Checklist for Professional UI
- [ ] **Accessibility**: Contrast ratios must be high. Touch targets min 44x44px.
- [ ] **Style Consistency**: Colors and spacing should follow a strict design token system.
- [ ] **Typography**: Use distinctive, characterful fonts (Outfit, Inter, Playfair) instead of system defaults.
- [ ] **Performance**: Animations must be 60fps. Avoid layout shifts.
- [ ] **Dark Mode**: High-contrast, non-muddy dark themes with proper elevation shadows.
