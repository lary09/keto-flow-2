# Frontend Design

## Design Thinking
- **Tone Choice**: Luxury/Refined, Minimalist, or Bold Maximalist. Stick to it.
- **Unforgettable Factor**: What is the one element the user will remember? (e.g., the 7-day plan card animation).

## Aesthetics Guidelines
- **Typography**: Pair a distinctive display font with a refined body font.
- **Color**: Dominant colors with sharp accents. Use CSS variables.
- **Motion**: Focus on high-impact moments (page loads, staggered reveals).
- **Backgrounds**: Use depth, mesh gradients, and grain overlays instead of flat solids.

## Constraints
- **Framework**: Next.js 15+ App Router.
- **Performance**: Zero CLS (Cumulative Layout Shift).
